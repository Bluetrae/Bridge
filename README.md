## 目录结构

- `Surge/`
  - `Configs/`：Surge 配置文件（模板/示例）
  - `Rules/`：Surge 规则文件（按功能分类）
- `Loon/`
  - `Configs/`：Loon 配置文件（模板/示例）
  - `Rules/`：Loon 规则文件（按功能分类）

## Rules 分类约定

- `Ads.list`：广告/追踪拦截与净化
- `Streaming.list`：流媒体相关规则
- `Social.list`：社交媒体相关规则

## 使用方式（最小）

1. 进入对应工具目录（Surge / Loon）
2. 从 `Configs/` 选择配置模板；从 `Rules/` 选择需要的规则文件
